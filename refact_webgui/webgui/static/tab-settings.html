Gated Models Token

Token is used to download gated (need authentication to download) models from Hugginface Hub. Obtain necessary token at Hugginface, accept terms and conditions if any and you can be able to add gated model at Model Hosting page.

Private Git Repositories keys

Generate new SSH key and open your Github Settings. Click "New SSH key" button, and paste generated SSH key. "Key Name" serves to differentiate one key from another.
